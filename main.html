Final exam:

In order to perform computer based quiz, you have to perform three steps:

  1. install lockdown browser
  2. make required configs
  3. go to Quizes folder to start the required quiz. Note that all quiz will be only available at a pre-announced time and also its time bound ( i.e. you have to complete it within a certain time duration).

All the steps are explained below for your guidance:

  1. Computer based quiz will require RESPONDUS LOCKDOWN BROWSER. You can download it from the following link:
    • Download the Respondus LockDown Browser for Texas Tech
    • View the LockDown Browser Student Quick Start Guide
  2. After installing the Lockdown browser, you have to configure your host file as follows:
    • White screen when trying to launch Respondus LockDown Browser
    • If you are not much familiar with host file configuration, please follow these specific steps.
      1. Windows Users: Close the LockDown Browser and do the following:
        • Open your local host file in NotePad (right click on Notepad and click Run as Administrator, open the following file: C:\Windows\System32\drivers\etc\hosts ).
        • Add the following to the bottom of the file: 129.118.242.3 fs.ttu.edu (Do not include the â€œ#â€ that is shown in the examples in the file)
        • Save the file
        • Windows 8 and above users:
          1. Save the file either on your desktop or in documents.
          2. Go to the location where the new file is saved.
          3. Right click
          4. Rename (if you do not see the extension of .txt click on view at the top of the page and be sure File name extensions is checked).
          5. Remove .txt and save
          6. Copy the file
          7. Go back to the hosts folder (C:\Windows\System32\drivers\etc\hosts)
          8. Paste the file and Replace the hosts file.
      2. Mac Users:
        1. Open Terminal
        2. type sudo nano /etc/hosts.
        3. Then, press the Return key.
        4. Enter the password of the current administrator account.
        5. Add the following entry to the bottom of the file: 129.118.242.3 fs.ttu.edu
        6. Save the file
  3. go to Quizes folder to start the required quiz. Note that all quiz will be only available at a pre-announced time and also its time bound ( i.e. you have to complete it within a certain time duration).
